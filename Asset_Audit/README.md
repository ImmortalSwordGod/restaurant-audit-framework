# Asset Audit

This module contains frameworks and procedures for physical asset verification and asset control.

## Objectives

- Verify existence of assets.
- Ensure proper asset tagging.
- Detect missing or damaged assets.
- Reconcile physical assets with records.

## Key Activities

- Physical Verification
- Asset Tag Validation
- Asset Reconciliation
- Disposal Verification

## Future Enhancements

- QR Code Tracking
- Barcode Asset Management
- Automated Asset Register
